# hand-Acupoints
這是一個手背穴位辨識的AI。
剛進入程式介面就如下。程式包含三個部分，首先點擊Acupoint Diagram，會顯示我們感測的六個手背穴位的介紹。
接著點擊Hand Detect，會開啟電腦前鏡頭或外接鏡頭(如圖十八)，按下Crop and Save Hand即可拍下圖片並裁切出物體部分及儲存圖片，然後點擊Point Show，就會顯示方才儲存的手部圖片並顯示穴位標點(如圖十九)。
